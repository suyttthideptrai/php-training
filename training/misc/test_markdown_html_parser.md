### Sử dụng `var_dump()` để kiểm tra kiểu dữ liệu và giá trị của biến
- Ví dụ:
```php
$age = 22;
$name = "ThanhNT";
$score = 8.5;
$hobby = ['Games', 'Coding'];
$mariage_status = null;
$is_super_man = false;
var_dump($age);
var_dump($name);
var_dump($score);
var_dump($hobby);
var_dump($mariage_status);
var_dump($is_super_man);
```
