# Các PHP Comments

1. Comment một dòng
- Sử dụng dấu `//` hoặc `#` để comment một dòng
```php
// Comment 1 dòng
# Comment 1 dòng
```
2. Comment nhiều dòng
- Sử dụng cặp dấu `/*` và `*/` để comment nhiều dòng
```php
    /* 
    Đây
    là
    comment 
    nhiều
    dòng
    */
```
```php
    /**
     * Đây là comment nhiều dòng
     * cũng có thể dùng để tạo PHPDoc
     */
```
